# 📇 Contact Book GUI

![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)
![Framework](https://img.shields.io/badge/framework-Tkinter-orange.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A sleek and efficient **Contact Management System** built with Python and Tkinter. This application provides a graphical interface to store, search, and manage your personal contacts with built-in data validation.

---

## ✨ Features
* **Add Contacts**: Save names, phone numbers, and email addresses.
* **Search Functionality**: Quickly find contacts by name.
* **Update & Edit**: Modify existing contact details on the fly.
* **Delete**: Remove outdated contacts with a single click.
* **Input Validation**: Ensures phone numbers and emails are formatted correctly before saving.
* **Sorted View**: Automatically displays your contact list in alphabetical order.
