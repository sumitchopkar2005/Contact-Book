📇 Contact Manager Application

A simple GUI-based Contact Manager built with Python and Tkinter.
This application allows users to add, search, update, and delete contacts with a clean and user-friendly interface.

🚀 Features
➕ Add new contacts with details
🔍 Search contacts by name
✏️ Update existing contact information
❌ Delete unwanted contacts
📜 View all stored contacts
✅ Input validation with error messages for incorrect entries
📊 Contacts can be sorted alphabetically by name

🛠️ Tech Stack

Python (Core Logic)
Tkinter (GUI Framework)

⚙️ Installation & Usage
Clone the repository
git clone https://github.com/your-username/contact-manager.git
cd contact-manager


Run the application
python contact_manager.py
